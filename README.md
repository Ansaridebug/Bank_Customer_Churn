# Bank_Customer_Churn
Bank Churn PredictionThis project develops a machine learning solution to identify at-risk bank customers. By analyzing demographics and financial behavior—such as credit score, account balance, and product usage—the model predicts the probability of a customer leaving the bank.

Bank Customer Churn Prediction using Machine Learning
A comprehensive predictive analytics project that identifies key factors influencing customer churn.Models:
Random Forest, XGBoost, and Logistic Regression.
Key Features: Credit Score, Tenure, Account Balance, and Activity Status.
Techniques: Data Cleaning, EDA, SMOTE for imbalanced data, and Feature Engineering.
